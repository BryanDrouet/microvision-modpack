# MicroVision Modpack - Guide de Téléchargement et Installation

## 📥 Télécharger le Launcher

Le launcher MicroVision est disponible à deux endroits:

### Option 1: Via le site web (RECOMMANDÉ)
1. Visite **https://github.com/BryanDrouet/microvision-modpack**
2. Clique sur le bouton "**⬇️ Télécharger MicroVision-Setup.exe**" sur la page d'accueil
3. Le fichier se télécharge (~50 MB)

### Option 2: Via GitHub Releases
1. Va sur: **https://github.com/BryanDrouet/microvision-modpack/releases**
2. Trouve la dernière release avec `MicroVision-Setup.exe`
3. Télécharge le fichier

## 🚀 Installation

### Étape 1: Exécuter l'installateur
1. Double-clique sur `MicroVision-Setup.exe`
2. Accepte l'accord de licence
3. Choisis ton dossier d'installation (par défaut: `C:\Users\[USER]\AppData\Local\MicroVision`)

### Étape 2: Téléchargement des mods
- Le launcher téléchargera automatiquement les 135 mods (~1.3 GB)
- Les mods proviennent du cloud Cloudflare R2
- **Première utilisation**: ~15-30 minutes (selon ta connexion)
- Les téléchargements suivants seront bien plus rapides

### Étape 3: Lancer le jeu
1. Ouvre le launcher MicroVision
2. Connecte-toi avec ton compte Minecraft (Microsoft/Mojang)
3. Clique sur "**PLAY**"
4. Profite! 🎮

## ⚡ Dépannage

### Le launcher ne s'ouvre pas
- Assure-toi d'avoir Java 17+: https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html
- Redémarre ton ordinateur
- Réinstalle le launcher

### Les mods ne téléchargent pas
- Vérifie ta connexion Internet
- Désactive temporairement ton VPN/proxy
- Regarde la barre de progression (elle peut être lente au départ)

### Le jeu crash au lancement
- Augmente la RAM du launcher (paramètres du launcher)
- Baisse les paramètres graphiques dans le jeu
- Mets à jour ton pilote GPU

### Vous avez d'autres problèmes?
- Ouvre une **issue** sur: https://github.com/BryanDrouet/microvision-modpack/issues
- Décris le problème en détail avec les logs
